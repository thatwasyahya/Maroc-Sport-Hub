# **App Name**: Maroc Sport Hub

## Core Features:

- Facility Mapping: Display sports facilities on a map using OpenStreetMap and Leaflet.js, including clustering of facilities.
- Excel Import: Enable admins to import facility data from Excel files, with validation and deduplication. Mock geocoding will be used.
- User Authentication: Implement local authentication using email/password with JWT.
- Booking System: Allow users to view facility availability and book time slots, with conflict checking on the backend.
- Admin Dashboard: Provide an admin interface for managing facilities, reservations, and user roles, including Excel import functionality.
- Mock Data Generation: Generate realistic mock data for facilities, users, and reservations using faker.js.
- Role-Based Access Control: Enforce role-based access control (super_admin, admin, user) on both the frontend and backend.

## Style Guidelines:

- Primary color: Vibrant orange (#FF7F50), evoking the energy of sports and Moroccan sunsets.
- Background color: Light peach (#FFEEDD), a soft and warm background that doesn't distract from the content.
- Accent color: Muted teal (#468499), providing a contrasting but harmonious complement to the primary color.
- Body and headline font: 'PT Sans', a humanist sans-serif offering both a modern look and a touch of warmth, suitable for headlines or body text
- Use clear and recognizable icons for different sports and facility types. Icons should be consistent with the overall design.
- Design a responsive layout that adapts to different screen sizes, ensuring a consistent user experience on desktop and mobile devices.
- Use subtle animations to enhance user interactions, such as transitions between pages and loading indicators.
